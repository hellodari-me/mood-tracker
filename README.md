# Мой ритм

Приватный адаптивный трекер настроения. Работает как PWA на ноутбуке и iPhone без публикации в App Store.

## Быстрый запуск на компьютере

Откройте папку в терминале и запустите локальный сервер:

```sh
python3 -m http.server 8080 --directory mood-tracker
```

Затем откройте `http://localhost:8080`.

## Установка на iPhone

Для установки на экран «Домой» приложение должно быть доступно по HTTPS. Опубликуйте содержимое папки `mood-tracker` на любом статическом хостинге (например, GitHub Pages, Netlify или Cloudflare Pages), затем:

1. Откройте ссылку в Safari.
2. Нажмите «Поделиться».
3. Выберите «На экран “Домой”».

## Хранение и перенос

Записи остаются в браузере конкретного устройства. В разделе «Данные» можно скачать JSON и импортировать его на другом устройстве без дубликатов. CSV подходит для Excel или Google Sheets.

Для автоматической синхронизации между устройствами потребуется подключить приватное облачное хранилище и вход по паролю.
